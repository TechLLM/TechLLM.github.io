## 🔑 개요

2026년 5월 19일, Google은 I/O 개발자 컨퍼런스에서 **Gemini Omni Flash**를 전격 출시했다. 단순한 영상 생성 모델 업데이트가 아니다. 텍스트·이미지·오디오·영상을 한꺼번에 입력으로 받아 하나의 일관된 영상으로 만들어내는, Gemini 지식 기반 위에 세워진 완전히 새로운 멀티모달 아키텍처다.

## 📌 핵심 요약

| 항목 | 내용 |
|---|---|
| 모델명 | Gemini Omni Flash |
| 출시일 | 2026년 5월 19일 |
| 최대 영상 길이 | 10초 (정책적 제한, 모델 한계 아님) |
| 입력 | 텍스트 + 이미지 + 오디오 + 영상 (조합 자유) |

| 워터마크 | SynthID 내장 |
| 무료 배포 | YouTube Shorts, YouTube Create |
| 유료 구독 | Gemini AI Plus ($7.99/월), Pro, Ultra |
| API | 수 주 내 개발자 공개 예정 |
| 상위 모델 | Omni Pro 계획 중 (출시일 미정) |

## 📝 본문

### 기존 Veo와 무엇이 다른가

Google은 이미 Veo라는 전용 영상 모델을 운영하고 있었다. 텍스트와 이미지를 입력받아 영상으로 변환하는 모델이다. Omni Flash는 그 연장선이 아니다.

**핵심 차이는 세 가지다.**

첫째, **입력 다양성**. Veo는 텍스트 또는 이미지 하나씩 받는 구조였다면, Omni Flash는 캐릭터 참조 이미지 + 음성 오디오 파일 + 원하는 조명이 담긴 영상 클립을 동시에 넣으면 그 세 조건을 모두 반영한 영상을 단번에 내놓는다. 각 입력을 붙이는 게 아니라, 모델이 입력들을 추론해 하나의 출력으로 합친다.

둘째, **대화형 편집**. Veo는 편집할 때마다 프롬프트를 다시 써서 영상을 새로 생성해야 했다. Omni Flash는 "조명을 따뜻하게 바꿔줘", "개를 고양이로 교체해" 같은 채팅 메시지 한 줄로 기존 클립을 수정한다. 나머지는 그대로 유지하면서.

셋째, **배포 전략**. YouTube Shorts와 YouTube Create에 무료로 내장했다. 개발자가 아닌 일반 사용자에게 즉시 도달하는 방식이다.

### 10초 제한은 기술적 한계가 아니다

발표 무대에서 Google이 직접 밝혔다. "모델 한계가 아니라 더 많은 사람 손에 빨리 쥐여주기 위한 선택"이라고. 대부분의 사용자가 당장은 긴 영상을 원하지 않는다는 판단도 있다. 정책이 풀리면 곧바로 더 긴 영상 생성이 가능해진다는 뜻이기도 하다.

### 오디오 편집은 의도적으로 잠겼다

Omni Flash는 오디오를 입력으로 쓸 수 있지만, **이미 생성된 영상 속 음성이나 오디오를 편집하는 기능은 제공하지 않는다**. 이유는 명확하다 — 선거 시즌 딥페이크 우려다. Google은 SynthID 워터마크를 모든 출력물에 심고, 아바타 생성 시 전용 온보딩을 거치게 해 오용을 막는 구조를 마련했다. 오디오 편집 제한은 정책과 탐지 기술이 안정화되면 풀릴 것으로 보인다.

### 실제 데모가 말해주는 것

발표 자리에서 공개된 예시 두 가지가 인상적이다. 단순 프롬프트 "단백질 접힘의 클레이메이션 설명 영상"을 넣자, 알파 헬릭스와 베타 시트를 설명하는 스톱모션 스타일의 나레이션 영상이 즉시 생성됐다. 구슬이 표면을 굴러가는 영상에서는 물리적으로 정확한 충격음까지 동기화됐다. 정밀한 물리 시뮬레이션과 멀티모달 이해력을 동시에 보여준 장면이었다.

## 💡 실무자가 볼 핵심 포인트

- **콘텐츠 크리에이터**: YouTube Shorts와 YouTube Create에 즉시 무료 통합. 지금 바로 써볼 수 있다.
- **개발자**: API는 수 주 내 공개 예정. 기업용 구독도 함께 지원 예정.
- **Veo 사용자**: Omni가 대체재가 아닌 보완재로 운영될 가능성 높음. Omni는 편집·멀티입력에 강점, Veo는 현재 API 생태계가 더 성숙.
- **딥페이크 우려**: SynthID 워터마크 + 아바타 온보딩 강제 → 안전장치 내장 확인.
- **로드맵**: Omni Pro(상위 모델) 계획 중. 더 긴 클립·더 높은 품질은 Pro 출시 이후.
