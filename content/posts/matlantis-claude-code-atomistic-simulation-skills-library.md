## 개요

원자 수준 시뮬레이션 플랫폼 Matlantis가 Claude Code 통합을 발표했다. 연구자가 자연어로 시뮬레이션을 생성·편집·실행할 수 있는 AI 에이전트 통합이다. Matlantis 전용 지식을 패키징한 스킬 라이브러리가 GitHub에 즉시 공개됐고, Matlantis 터미널 내부에서 Claude Code를 직접 실행하는 인스톨러도 이후 업데이트에서 출시 예정이다.

---

## 핵심 요약

- **Claude Code 내장**: Matlantis 터미널에서 Claude Code를 직접 실행하는 인스톨러 출시 예정
- **GitHub 스킬 라이브러리**: 구조 완화·분자동역학·반응 경로 탐색·결정 구조 예측 등 포함, 즉시 사용 가능
- **자연어 시뮬레이션**: 코드 작성 없이 대화로 시뮬레이션 생성·실행·해석 가능
- **비전문가 접근성 확대**: 프로그래밍 역량 없는 실험 연구자도 원자 시뮬레이션 도구 활용 가능
- **도입 규모**: 촉매·배터리·반도체·합금 등 150개 이상 기업·기관 사용 중
- **공동 개발**: PFN(Preferred Networks)·ENEOS·미쓰비시상사 공동 투자

## 본문

### 원자 시뮬레이션의 세 가지 장벽

Matlantis CEO 오카노하라 다이스케는 원자 시뮬레이션 접근성의 세 가지 장벽을 이렇게 정의했다. 인프라, 과학적 지식, 그리고 스크립팅이다.

Matlantis는 지난 5년간 인프라 장벽을 제거했다. PFP(Preferred Potential) AI 모델을 클라우드 서비스로 제공해 대규모 계산 환경 구축 부담을 없앴다. 기존 물리 시뮬레이터에 딥러닝 모델을 통합해 시뮬레이션 속도를 수만 배 높였다.

이번 통합이 겨냥하는 것은 세 번째 장벽, 즉 스크립팅이다. 연구자의 질문과 실제 작동하는 시뮬레이션 사이에 놓인 코딩 레이어다.

### GitHub 스킬 라이브러리: 즉시 사용 가능

GitHub에 공개된 스킬 라이브러리는 Matlantis 전용 지식을 에이전트가 참조할 수 있는 형식으로 패키징한다. 함수, API, 대표적인 워크플로우가 포함된다.

에이전트 기반 모델의 일반 학습 데이터에는 Matlantis 플랫폼의 전문 지식이 없다. 스킬 라이브러리가 이 공백을 메운다. 에이전트가 처음 시도에서 더 정확하고 맥락에 맞는 시뮬레이션 스크립트를 생성할 수 있게 된다.

현재 포함된 초기 워크플로우는 다음과 같다.

- **구조 완화(structure relaxation)**: 에너지가 최소화된 안정 구조 탐색
- **분자동역학(molecular dynamics)**: 시간에 따른 원자 운동 시뮬레이션
- **반응 경로 탐색(reaction pathway exploration)**: 화학 반응의 전이 상태·활성화 에너지 계산
- **결정 구조 예측(crystal structure prediction)**: 새로운 소재의 결정 구조 예측
- **시각화**: 시뮬레이션 결과 렌더링
- **외부 데이터베이스 구조 검색**: 기존 소재 데이터베이스 연동

스킬 라이브러리는 고객 사용 사례가 쌓이면서 계속 확장될 예정이다.

### Claude Code 터미널 내장: 워크플로우 단절 없이

인스톨러 출시 이후 연구자는 Matlantis 터미널에서 Claude Code를 직접 실행할 수 있다. 인컨텍스트로 에이전트를 실행한다는 것이 핵심이다.

기존 방식에서는 시뮬레이션 설계, 스크립트 작성, 계산 실행, 결과 해석이 각각 다른 도구와 환경에서 이뤄졌다. 이번 통합 이후의 흐름은 다르다. 시뮬레이션하려는 내용을 자연어로 설명하면, 에이전트가 스크립트를 생성하거나 편집하고, 계산을 실행하고, 결과를 해석한다. 모두 워크플로우를 벗어나지 않고 이뤄진다.

### 연구 역할별로 달라지는 효과

통합이 가져오는 변화는 연구자 유형에 따라 다르게 나타난다.

**실험 연구자**: 코드 대신 대화로 시뮬레이션을 생성하고 결과를 해석할 수 있다. 계산화학 전공이나 프로그래밍 역량 없이도 원자 시뮬레이션을 실질적인 도구로 쓸 수 있게 된다.

**계산화학 전문가**: 발표된 분석 재현, 후보 물질 스크리닝, 새로운 시스템에 맞춘 워크플로우 조정에 드는 반복적 스크립팅 작업이 줄어든다. "먼저 시도하고 분석하는" 방식으로 전환이 가능하다.

**R&D 리더**: 전문가 투입 시간을 정당화하기 어려웠던 주제에도 계산적 방법을 적용할 수 있다. 연구 범위가 넓어진다.

### 산업 AI 에이전트 통합의 방향

Matlantis의 움직임은 더 넓은 흐름을 반영한다. AI 에이전트가 실험 단계를 지나 기술 워크플로우 프로덕션 사용으로 전환하고 있다. Claude Code를 플랫폼에 내장하고 도메인 전용 스킬 라이브러리를 함께 제공하는 방식은 일반 에이전트 역량과 특수 도메인 절차·API를 연결하는 구체적인 방법론이다.

150개 이상 기업이 촉매, 배터리, 반도체, 합금, 윤활제, 세라믹, 화학 소재 발견에 Matlantis를 사용한다. 이 사용자 기반에 에이전트 인터페이스가 추가된다.

## 실무자가 볼 핵심 포인트

- **스킬 라이브러리 구조 참고**: 도메인 전용 지식을 에이전트에 공급하는 방식이 구체화됐다. Matlantis가 공개한 GitHub 스킬 라이브러리는 자체 서비스에 유사한 에이전트 통합을 설계할 때 참고 모델로 쓸 수 있다.
- **인컨텍스트 실행 방식**: 에이전트를 외부에서 호출하는 것이 아니라 작업 환경 안에서 직접 실행하는 구조가 핵심이다. API 연동 방식 대비 워크플로우 단절이 적다.
- **비전문가 접근성 확대 신호**: 소재·화학 R&D 분야에서 비코딩 연구자의 시뮬레이션 접근성이 실질적으로 높아진다. 이 분야 소프트웨어를 개발하거나 사용하는 팀이라면 주목할 변화다.
- **현재 사용 가능 vs 예정**: 스킬 라이브러리는 GitHub에서 지금 바로 사용할 수 있다. Claude Code 터미널 내장 인스톨러는 이후 업데이트에서 출시 예정이므로 타임라인을 확인할 것.
- **PFP 모델 맥락**: Matlantis의 핵심은 PFP(Preferred Potential) 범용 원자 잠재력 모델이다. 에이전트 통합은 이 모델 위에 쌓인 접근성 레이어다. 정확도 자체는 기존과 동일하다.
