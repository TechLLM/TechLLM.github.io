AI 영상 플랫폼에 월 $20~30 내고 있다면, 잠깐 멈춰봐.

요즘 AI 이미지·영상 생성 툴들, 쓸 만한 건 다 유료야. Midjourney, Runway, Pika... 하나씩 구독하다 보면 어느새 월 몇만 원이 그냥 증발하주. 그런데 이걸 전부 무료로, 그것도 오픈소스로 해결할 수 있는 프로젝트가 나왔어.

이름하여 **Open Generative AI**.

---

## 이게 뭔데?

한 줄로 요약하면, **200개 넘는 AI 모델을 한 곳에서 무료로 쓸 수 있는 오픈소스 스튜디오**야.

Flux, Midjourney 스타일 모델, Kling, Sora, Veo, Seedream, Wan 2.2, Hunyuan Video, LTX Video... 이름만 들어도 아는 굵직한 모델들이 다 들어가 있어. 텍스트→이미지, 이미지→이미지, 텍스트→영상, 이미지→영상, 심지어 입술 싱크(Lip Sync)까지.

구독료 없고, 벤더 종속도 없어. 그냥 내 컴퓨터에서 돌리는 거야.

---

## 주요 기능들

크게 다섯 개 스튜디오로 나뉘어 있어:

**🎨 Image Studio**
- 텍스트 → 이미지: 50개 넘는 모델 지원
- 이미지 → 이미지: 55개 넘는 변환 모델
- 최대 14개 참고 이미지 동시 입력 가능 (멀티 이미지 합성에 이거 진짜 강력해)

**🎬 Video Studio**
- 텍스트 → 영상: 40개 이상 모델
- 이미지 → 영상: 60개 이상 모델

**💋 Lip Sync Studio**
- 초상화 이미지 + 오디오 파일 → 말하는 영상
- 9개 모델 지원

**🎥 Cinema Studio**
- 영화적 카메라 앵글, 움직임 컨트롤 가능
- 프로 수준의 씬 연출 지원

**🔧 Workflow Studio**
- 노드 기반 시각적 AI 파이프라인 빌더
- n8n 같은 구조로 AI 작업들을 연결해서 자동화 가능

---

## 로컬 추론 — Mac 쓰는 사람한테 특히 좋아

이게 진짜 포인트인데, API 키 없이도 **내 기기에서 직접 추론**할 수 있어.

- **macOS (M1~M4)**: `sd.cpp` 엔진 + Apple Silicon Metal 가속 지원. M4 맥북으로 이미지 생성하면 속도 꽤 나와.
- **CUDA GPU 서버 (Windows/Linux)**: `Wan2GP` 엔진으로 GPU 가속 지원

그러니까 클라우드에 데이터 안 올라가고, 내 머신 안에서 다 처리돼. 프롬프트든 이미지든 영상이든 전부 로컬에 남아. 개인 작업할 때 프라이버시 걱정 없이 쓸 수 있어.

---

## 필터 없음, 가이드레일 없음

이건 논란 될 수도 있지만 사실로 명시되어 있어 — **콘텐츠 필터, 가이드레일, 프롬프트 거부가 없어**.

창작의 자유를 완전히 사용자한테 돌려준다는 철학이야. 물론 그 자유에는 책임이 따르주만, 상업 플랫폼들이 너무 보수적인 필터로 원하는 걸 못 만들게 막는 경험 해봤다면 이게 얼마나 큰 차이인지 알 거야.

---

## 우리 워크플로에 붙이면 좋은 점

개인적으로 이 프로젝트가 흥미로운 이유는 **Workflow Studio** 때문이야.

노드 기반으로 AI 파이프라인을 시각적으로 구성할 수 있다는 게, 단순히 "이미지 생성 툴"에서 "AI 자동화 도구"로 격상시켜줘. 예를 들면:

- 블로그 포스트 주제 → 텍스트 프롬프트 → 이미지 생성 → Lip Sync 영상화 → 자동 저장

이런 흐름을 노드로 연결해서 반자동화할 수 있어. 콘텐츠 제작 파이프라인에 붙이면 꽤 쓸모 있을 거야.

---

## 마무리

유료 AI 영상 툴들이 구독 장벽을 높이는 동안, 오픈소스 커뮤니티는 조용히 이런 걸 만들고 있었어. 200개 넘는 모델을 무료로, 로컬에서, 필터 없이 쓸 수 있다는 건 — 특히 크리에이터나 개발자한테는 무시하기 어려운 옵션이야.

당장 모든 작업을 옮겨갈 필요는 없어도, 한번 세팅해서 써보는 건 충분히 가치 있어. macOS M 시리즈 있으면 Metal 가속 덕에 이미지 생성 속도도 나름 쓸 만할 거고.
