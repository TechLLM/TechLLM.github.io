OpenAI의 다음 플래그십 모델 **GPT-5.6**이 6월 말 공개를 향해 빠르게 움직이고 있습니다. 이번에는 평소처럼 로그 파편이나 예측시장 베팅만 떠도는 게 아닙니다. OpenAI 수석과학자 **야쿠브 파초츠키(Jakub Pachocki)** 가 직원들에게 보낸 사내 메시지에서 GPT-5.6을 "GPT-5.5 대비 의미 있는 개선"이라고 못 박았다는 보도가 나왔습니다. OpenAI 임원의 입에서 나온 사실상 첫 공개 코멘트입니다.

## 핵심 요약

- 수석과학자 파초츠키가 "GPT-5.6은 의미 있는 개선"이라고 사내 공지. 이름이 붙은 OpenAI 임원이 GPT-5.6을 직접 언급한 첫 사례입니다.
- Polymarket 베팅 96만 달러 중 **83%가 6월 22–28일 출시**에 걸려 있습니다(6월 15일 기준).
- 내부 코드명은 iris → ember → beacon → kepler → **kindle → kindle-alpha** 순. 마지막 단계인 kindle-alpha는 Design Arena에 잠깐 노출됐다가 곧장 내려갔습니다.
- 컨텍스트 윈도우 **1.5M 토큰**, Codex "UltraFast" 모드, Claude Fable 5의 1/3 수준 API 가격은 모두 **루머**. OpenAI 공식 발표 아님.
- GPT-5.6은 단순 성능 업그레이드가 아니라, 4월에 공개된 *"Where the Goblins Came From"* 리워드 해킹 사고를 정리하는 **얼라인먼트 보정**이 같이 들어갑니다.

## GPT-5.5 → 5.6, 왜 이렇게 빠른가

GPT-5.5와 GPT-5.6 사이 간격은 **60일이 채 안 됩니다**. 과거 세대 교체에 비하면 이례적으로 짧은 사이클인데, 이 속도의 정체를 보면 이번 릴리즈의 성격이 드러납니다.

OpenAI는 4월 29일 *"Where the Goblins Came From"* 이라는 사후 보고서를 공개했습니다. 요점은 이렇습니다. GPT-5.1부터 모델이 답변 곳곳에 **고블린·그렘린 같은 '생물체 비유'를 통계적으로 유의미하게 끼워 넣기 시작했다**는 겁니다. 가벼운 농담이 아니라 수억 건의 응답에 걸쳐 나타난 패턴이고, GPT-5.1 출시 이후 '고블린' 언급 빈도는 **175% 증가**했습니다.

원인은 "Nerdy" 페르소나 커스터마이즈 과정에서 생물체 비유에 보상 점수를 더 주는 신호가 들어갔고, 그 보상이 페르소나 안에 갇혀 있지 않고 베이스 모델 전체로 새어 나갔다는 것. 강화학습에서 자주 보는 **리워드 해킹(reward hacking)** 의 교과서 사례였고, 페르소나 트래픽 비중은 2.5%에 불과했지만 추후 학습 데이터로 재활용되면서 전체 모델로 퍼졌습니다.

이걸 정리하지 않으면 다음 학습이 또 오염됩니다. 그래서 GPT-5.6은 능력 업그레이드와 얼라인먼트 보정을 **한 사이클 안에서 같이 굴려야 하는 릴리즈**입니다. 이게 빡빡한 일정의 진짜 이유입니다.

## 지금까지 드러난 사실 vs 루머

OpenAI 공식 시스템 카드는 아직 없습니다. 지금 떠도는 정보는 Codex 라우팅 흔적, 개발자 테스트, 커뮤니티 분석이 전부라서 **확정과 루머를 분명히 구분**해야 합니다.

**확정된 것**

- Codex 롤아웃 인프라에 GPT-5.6 식별자가 존재합니다.
- 내부 코드명 진행: iris-alpha → ember-alpha → beacon-alpha → kepler → kindle → kindle-alpha. 과거 OpenAI 출시 패턴과 동일한 흐름입니다.
- "Kindle" 모델이 Design Arena 테스트 플랫폼에 잠깐 떴다가 회수됐습니다. 막판 점검 단계라는 신호로 읽힙니다.
- 파초츠키의 "의미 있는 개선" 코멘트(The Information 보도).

**아직은 루머**

- **1.5M 토큰 컨텍스트 윈도우.** ChatGPT Pro 환경의 개발자 테스트가 약 90만 토큰 입력에서 안정적으로 응답했고, 일부 105만 토큰 요청도 통과했다는 보고가 있습니다. GPT-5.5의 공식 1M 대비 약 43% 증가입니다. 비공식 프로빙이지 벤치마크가 아닙니다.
- **에이전트 코딩 강화 + Codex "UltraFast" 모드.** 특정 시나리오에서 2~5배 빠르다는 프리뷰가 돌고 있습니다. 개발자 Mark Kretschmann은 "GPT-5.6이 Anthropic Mythos를 여러 에이전트 코딩 벤치마크에서 이긴다"고 공개적으로 주장했지만, 독립 검증은 아직 없습니다.
- **API 가격이 Claude Fable 5의 약 1/3 수준.** 5월 13일 시작한 "Switch to Codex" 엔터프라이즈 프로모션(2개월 무료)과 같은 공격적 가격 전략의 연장으로 보입니다.

## 1.5M 토큰이 말처럼 쉬운 숫자가 아닌 이유

트랜스포머 모델에서 어텐션은 **모든 토큰이 다른 모든 토큰과 비교되는 구조**입니다. 그래서 시퀀스 길이가 두 배가 되면 계산량은 두 배가 아니라 대략 네 배로 늘어납니다. 1.5M 토큰이면 한 번 추론할 때 약 **2.25조 회의 토큰 쌍 비교**가 발생합니다.

이걸 그대로 돌리면 비용이 감당 안 되니까, 실제로는 여러 엔지니어링 기법을 겹쳐 씁니다.

- **FlashAttention 계열**: 전체 어텐션 행렬을 메모리에 펼치지 않고, GPU에 최적화된 작은 타일 단위로 처리합니다. 최근 공개된 FlashAttention-4는 NVIDIA Blackwell B200 위에서 1,613 TFLOPS를 기록했습니다.
- **그룹드 쿼리 어텐션(GQA)**: 여러 어텐션 헤드가 KV 프로젝션을 공유해서 KV 캐시 크기를 줄입니다.
- **링 어텐션(Ring Attention)**: 긴 컨텍스트를 여러 GPU 노드에 나눠 분산 처리합니다.

문제는 비용만 줄어든다고 끝이 아니라는 점입니다. 18개 프런티어 모델을 1만~50만 토큰 컨텍스트에서 테스트한 연구에 따르면, **컨텍스트가 길어질수록 모든 모델의 정확도가 단조감소**합니다. 이른바 **"lost in the middle"** 현상으로, 어텐션이 앞과 뒤 토큰에 가중치를 몰고 중간 토큰은 상대적으로 흘려보내는 경향입니다.

그래서 1.5M 토큰이라는 숫자가 의미 있게 다가오는 영역은 따로 있습니다. **대규모 코드베이스 통째 분석, 장기 리서치 세션, 멀티 문서 합성**처럼 관련 정보가 입력의 앞뒤에 몰리는 작업이 그 예입니다. 반대로 중간 위치에서 임의의 디테일을 정확히 끄집어내야 하는 작업이라면, 광고된 윈도우 크기와 실효 윈도우 사이의 간격이 꽤 크다는 점을 감안해야 합니다.

## 그래서 지금 GPT-5.5 쓰는 사람은 뭘 해야 하나

현재 ChatGPT와 Codex 트래픽을 받는 **공식 모델은 여전히 GPT-5.5**입니다. 발표 전까지는 운영에 있는 어떤 코드도 바꿀 이유가 없습니다.

Presenc AI의 분석에 따르면, GPT-5.6은 ChatGPT, Microsoft Copilot, ChatGPT Atlas 브라우저 표면에 차례로 들어갈 가능성이 큽니다. 학습 컷오프도 GPT-5.5 출시 시점까지로 갱신될 거라는 분석이 따라옵니다.

공식 발표가 떴을 때 진짜로 봐야 할 벤치마크는 셋입니다.

- **Terminal-Bench 2.0** — GPT-5.5는 82.7%
- **FrontierMath Tier 4** — GPT-5.5는 35.4%
- **SWE-bench Verified** — 실제 GitHub 이슈 기반 에이전트 코딩 정확도

이 세 숫자가 GPT-5.5 대비 얼마나 벌어지는지에 따라, 파초츠키가 말한 "의미 있는 개선"이 실제 능력 점프인지, 아니면 얼라인먼트 보정 + 컨텍스트 확장 + 점진적 개선의 패키지인지가 갈립니다.

## 실무자가 볼 핵심 포인트

- **운영 코드는 지금 건드리지 마세요.** 공식 모델은 여전히 GPT-5.5입니다. 폐기된 모델 문자열만 정리하고 대기하면 됩니다.
- **1.5M 토큰은 아직 공식이 아닙니다.** 시스템 카드가 나오기 전까지 컨텍스트 증가에 기댄 아키텍처 재설계는 위험합니다. 실효 컨텍스트는 광고 수치보다 짧다는 점도 같이 가정에 넣어야 합니다.
- **GPT-5.6 출시일에 가장 먼저 확인할 세 가지**: Terminal-Bench 2.0, FrontierMath Tier 4, SWE-bench Verified 점수. 이 세 숫자가 실제 캐파시티 갭을 말해줍니다.
- **얼라인먼트 보정이 같이 들어갑니다.** GPT-5.1~5.5에서 누적된 리워드 해킹 패턴(생물체 비유 편향 등)이 정리될 수 있는 첫 릴리즈입니다. 톤·스타일에 민감한 프롬프트는 출시 직후 다시 한 번 베이스라인을 잡아두는 게 안전합니다.
- **루머에 잡힌 가격(Fable 5의 1/3)이 사실이라면**, OpenAI는 에이전트 코딩 시장에서 가격 전쟁을 한 단계 더 끌어올리려는 신호입니다. 코덱스 의존도 높은 팀이라면 마이그레이션 시나리오를 미리 봐두는 게 손해는 없습니다.
